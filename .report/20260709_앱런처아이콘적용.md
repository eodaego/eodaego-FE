# 앱 런처 아이콘 적용 보고서

## 변경 개요

`flutter_launcher_icons` 패키지를 사용해 원본 로고(`assets/app_logo_1024.png`, 1024×1024)로부터 앱 런처 아이콘을 생성하여 Android와 iOS에 일괄 적용했다.

- 원본 이미지: `assets/app_logo_1024.png` (1024px)
- Android: `mipmap-*` 밀도별 `ic_launcher.png` 5종 생성
- iOS: `AppIcon.appiconset` 전 사이즈 생성 (신규 50/57/72 사이즈 포함), iOS 요구사항에 따라 알파(투명도) 채널 제거

## 영향 범위

앱 런처 아이콘만 변경. 런타임 로직, 화면, 상태 관리 등과 무관하다.

## 변경 파일 목록

| 경로 | 종류 | 비고 |
| --- | --- | --- |
| `pubspec.yaml` | 설정 | `flutter_launcher_icons` 블록 및 assets에 `app_logo_1024.png` 등록 |
| `assets/app_logo_1024.png` | 원본 | 신규 원본 로고 (1024×1024, 약 787KB) |
| `android/app/src/main/res/mipmap-mdpi/ic_launcher.png` | 생성물 | Android 아이콘 |
| `android/app/src/main/res/mipmap-hdpi/ic_launcher.png` | 생성물 | Android 아이콘 |
| `android/app/src/main/res/mipmap-xhdpi/ic_launcher.png` | 생성물 | Android 아이콘 |
| `android/app/src/main/res/mipmap-xxhdpi/ic_launcher.png` | 생성물 | Android 아이콘 |
| `android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png` | 생성물 | Android 아이콘 |
| `ios/Runner/Assets.xcassets/AppIcon.appiconset/Contents.json` | 설정 | 아이콘 세트 인덱스 (신규 사이즈 항목 반영) |
| `ios/Runner/Assets.xcassets/AppIcon.appiconset/Icon-App-*.png` | 생성물 | iOS 아이콘 전 사이즈 (20/29/40/**50/57/72**/60/76/83.5/1024) |
| `ios/Runner.xcodeproj/project.pbxproj` | 설정 | `SETCATALOG_COMPILER_APPICON_NAME = AppIcon` 참조 |

## 설정 상세 (pubspec.yaml)

```yaml
flutter_launcher_icons:
  image_path: "assets/app_logo_1024.png"
  android: true
  ios: true
  remove_alpha_ios: true
```

## 재생성 방법

```bash
dart run flutter_launcher_icons
```

## 확인 사항 / 주의점

- 기기에서 아이콘 캐시가 남아 있을 수 있으므로, 앱 삭제 후 재설치하여 새 아이콘 반영을 확인한다.
- iOS는 `remove_alpha_ios: true`로 알파 채널을 제거하므로 투명 배경이 없다. 원본 로고에 투명 영역이 있으면 흰색 등으로 채워진 상태로 표시된다.
- 아이콘 파일은 도구 생성물이므로 원본(`app_logo_1024.png`) 또는 설정 변경 시 재생성해야 하며 수동 편집하지 않는다.
