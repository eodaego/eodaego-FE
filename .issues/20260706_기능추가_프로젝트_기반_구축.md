## 본문

## 📝 현재 문제점

- 어대GO 앱의 기반 코드가 없어, 화면과 기능 개발을 시작할 공통 토대(폴더 구조, 네트워크, 인증, 주요 서비스)가 필요합니다.

## 🛠️ 해결 방안 / 제안 기능

- Feature-first + Clean Architecture 구조(lib/ 하위 core·features·router)로 앱 기반을 세팅합니다.
- 코어 인프라부터 인증, 주요 서비스 레이어까지 구축해 이후 화면·기능이 바로 얹힐 수 있게 합니다.

## ⚙️ 작업 내용

- 코어 상수·config·errors·utils (디자인 토큰, EnvConfig, AppException, URL 유틸)
- 네트워크·스토리지 (Dio, AuthInterceptor, SecureTokenStorage, 401 자동 재발급)
- 공용 다이얼로그 + 점검/강제 업데이트 페이지
- 인증 (Google/Apple 로그인 → 백엔드 JWT), 유저 (닉네임·약관·회원탈퇴)
- 라우터 + 인증 페이지 뼈대 + 홈 플레이스홀더, main.dart 부트스트랩
- FCM·디바이스 서비스, Remote Config (버전 체크·강제 업데이트·점검), 위치·권한 서비스
