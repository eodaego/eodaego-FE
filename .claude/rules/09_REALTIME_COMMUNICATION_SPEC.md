## 1. 소켓 연결 요청

| **항목** | **값**                               | **비고**                                            |
| ------------ | -------------------------------------- | ----------------------------------------------------- |
| **URL**      | `ws://{도메인}/game-connection`    |                                                       |
| **Header**   | `Authorization: Bearer {Access_Token}` | **필수** (토큰 만료 시 재연결 필요) |
| **Type**     | STOMP                                  |                                                       |

## 2. 참고 사항

- 모든 시간 관련 데이터는 ISO-8601 표준 포맷으로 응답됩니다.
  `(2026-01-26T14:30:15.123456789+09:00)`
- 실시간 통신중 발생하는 오류는 API 오류와 같은 구조를 응답합니다.
  [STOMP ERROR 응답]

  ```json
  **{
    "title": "인증 만료",
    "status": 401,
    "detail": "인증정보가 만료되었습니다.",
    "instance": "STOMP"
  }**
  ```

  - 웹 소켓 연결 시 토큰이 만료된 경우 위와 같은 메세지를 응답합니다.
  - **instance의 필드 값이 STOMP인 에러들은 STOMP 프로토콜의 에러 응답이므로, (비즈니스 예외가 아닌) 해당 에러 응답을 받게 되면 웹 소켓 연결이 끊기게 됩니다.**
  - STOMP ERROR 프레임

- 비즈니스 로직에 대한 오류는 아직 구현이 안된 상황입니다.

## 3. 채팅

### **(1) 메세지 전송 (Publish)**

- **경로:** `/publish/game/{gameId}/chat`
- **특징:** 목적지(팀/전체)를 경로로 구분하지 않고, JSON Body의 `scope`로 구분합니다.
- **Payload (Request)**
  ```json
  {
    "message": "1층으로 와주세요!",
    "scope": "TEAM" // "TEAM" 또는 "ALL"
  }
  ```

### **(2) 메세지 수신 (Subscribe)**

- 클라이언트는 두 경로를 모두 구독해야 채팅 수신이 가능합니다.

| **수신 타입**  | **구독 경로 (Destination)**      | **설명**                                                            |
| -------------------- | -------------------------------------- | ----------------------------------------------------------------------- |
| **팀 채팅**     | `/subscribe/game/{gameId}/chat/{team}` | `{team}`: `police` 또는 `robber` (본인 팀만 구독 가능) |
| **전체 채팅**` | `/subscribe/game/{gameId}/chat/all`    | 게임 참가자 전원 수신                                     |

- **Payload (Response)**
  ```json
  {
  	"id" : "uuid-uuid-uuid...."
  	"gameId" : 1
    "sender": {
      "participantId": 5,
      "nickname": "창희도둑",
      "team": "ROBBER"
    },
    "message": "경찰 떴다!",
    "timestamp": "2026-01-26T14:30:15.123456789+09:00",
    "scope": "TEAM" // "TEAM" 또는 "ALL"
  }
  ```

### **(3) 오류 응답**

1. 본인의 소속이 아닌 채팅에 대한 구독 요청 시
   - **Payload (Response)**
     ```json
     {
       "title": "구독 권한 없음",
       "status": 401,
       "detail": "해당 팀 전용 채널을 구독할 권한이 없습니다.",
       "instance": "STOMP"
     }
     ```

### (4) **참고 - 데이터 흐름**

1. **Client(Connect)**: `ws://{도메인}/game-connection` 으로 웹 소켓 연결 요청
2. **Client(Pub):** `/publish/game/{id}/chat`으로 메시지 전송 (목적지 구분 없음)
3. **Server(Routing):** 메시지의 `scope`(ALL, TEAM)와 유저 정보(Team)를 확인하여 적절한 **Redis Topic** 결정
4. **Redis(Broker):** `game:{id}:chat:{team}` 토픽으로 메시지 발행
5. **Server(Sub):** 해당 토픽을 구독 중인 서버가 메시지 수신
   (여기서 의미하는 구독은 서버입장에서의 구독)
6. **Client(Sub):** 최종적으로 `/subscribe/game/{id}/chat/{team}`을 구독 중인 클라이언트에게 전달

## 4. 게임 로비

### **(1) 이벤트 전송**

로비의 상태 변경은 클라이언트의 HTTP PATCH 요청으로 시작됩니다.(자세한 명세는 스웨거에 있어요!) 서버는 이를 처리한 후 웹소켓으로 이벤트를 브로드캐스팅합니다.

- **팀 변경**: `PATCH /api/games/{gameId}/lobby/team`
- **준비 상태 변경**: `PATCH /api/games/{gameId}/lobby/ready`
- **특징**: 클라이언트가 직접 `/pub` 경로로 메시지를 쏘는 것이 아니라, API 호출 결과에 따라 서버가 이벤트를 발행합니다.

### **(2) 이벤트 수신**

클라이언트는 로비 입장 시 아래 경로를 구독하여 실시간 상태 변경을 수신합니다.

- **구독 경로**: `/sub/game/{gameId}/lobby`
- **Payload (Response)**:

```jsx
{
  "eventId": "uuid-random-string",
  "gameId": 1,
  "type": "TEAM_UPDATE",    // ENTER, EXIT, TEAM_UPDATE, READY_UPDATE, HOST_CHANGED, GAME_START
  "timestamp": "2026-02-02T14:10:00",
  "data": { ... }            // 상세 데이터 (타입별 상이)
}
```

### **(3) 주요 이벤트 데이터 필드 상세**

**요약**

| **이벤트 종류 (type)** | **발생 시점**                                                               | **데이터 핵심 포함 내용**                                                        |
| ----------------------------- | ---------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| **`ENTER`**                   | 새로운 유저가 방에 입장했을 때                                  | 입장 유저 상세 정보, 현재 방 인원 현황                               |
| **`EXIT`**                    | 유저가 방에서 나갔을 때                                               | 퇴장 유저 ID, 남은 인원 현황                                                 |
| **`TEAM_UPDATE`**             | 유저가 팀을 변경했을 때                                            | 변경된 유저 상세 정보, 팀별 인원 수, 준비 상태 초기화 정보 |
| **`READY_UPDATE`**            | 유저가 준비 버튼을 눌렀을 때                                     | 해당 유저 상세 정보                                                                |
| **`HOST_CHANGE`**             | 기존 방장이 나가거나 위임(아직은 없는 기능)했을 때 | 새 호스트 유저 상세 정보                                                          |
| **`GAME_START`**              | 모든 인원 준비 완료 시                                               | 게임 시작 신호 (추가 데이터 최소화)                                          |
| **`AREA_UPDATED`**            | 방장이 게임 영역 수정 버튼을 눌렀을 때                  | 새로운 영역 정보                                                                   |
| **`SETTINGS_UPDATED`**        | 방장이 게임 설정 수정 버튼을 눌렀을 때                  | 새로운 설정 정보                                                                   |
| **`KICKED`**                  | 방장이 특정 참가자 강제 퇴장 시킬 때                        |                                                                                               |

### (4) **참고 - 데이터 흐름**

- **Client(Connect)**: `ws://{도메인}/game-connection` 으로 웹 소켓 연결 요청
- **Client(API Request)**: HTTP PATCH 메서드로 팀 변경(`/team`) 또는 준비 상태 변경(`/ready`) 요청
- **Server(Business Logic)**: DB 트랜잭션 수행
- **Server(Transaction Commit)**: DB 커밋이 최종 성공한 시점에만 내부 이벤트 핸들러(`LobbyEventHandler`) 작동
- **Redis(Broker)**: `game:{id}:lobby` 토픽으로 업데이트된 로비 이벤트 메시지 발행
- **Server(Sub)**: 해당 토픽을 구독 중인 서버들이 메시지 수신 (클러스터링 환경 동기화)
- **Client(Sub)**: 최종적으로 `/subscribe/game/{id}/lobby`를 구독 중인 모든 클라이언트에게 변경된 상태 브로드캐스팅

## 5. 게임 이벤트

## 1. 이벤트 전송 개요

- 이벤트 트리거: 클라이언트 HTTP 요청
- 서버 처리: 유효성 검사 및 DB 업데이트 (트랜잭션)
- 트랜잭션 성공 시 이벤트 Publish
- 목적: 게임 상태 실시간 동기화

| **기능**                     | **Client Action**       | **URI**                             | **비고**                                           |
| ------------------------------- | ----------------------- | ----------------------------------- | ---------------------------------------------------- |
| 체포 요청                  | HTTP 요청            | `/api/games/{gameId}/system/arrest` | 경찰이 도둑 체포 버튼 클릭 시      |
| 탈옥 요청                | HTTP 요청            | `/api/games/{gameId}/system/escape` | 수감된 도둑이 탈옥 버튼 클릭 시 |
| 이벤트 전송 (나머지) | 없음 (HTTP 요청) | -                                   | 서버 내부 처리 후 발행                    |
| 이벤트 수신              | Sub                     | `/subscribe/game/{gameId}/system`   | 게임 참여자 전체 구독                   |
